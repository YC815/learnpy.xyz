## 1. 基本的 if 語句

`if` 讓程式根據條件執行不同的程式碼：

<TryIt
id="basic-if"
starter={`age = 18

if age >= 18:
    print("你已經成年了！")
    print("可以投票了")

print("程式結束")`}
enableAiReview
/>

**重點**：注意縮排（indentation）！Python 用縮排來區分程式碼區塊。

## 2. if-else：二選一

當條件不成立時執行另一段程式碼：

<TryIt
id="if-else"
starter={`score = 75

if score >= 60:
    print("及格！")
else:
    print("不及格，要補考")
    
print(f"你的分數是 {score}")`}
enableAiReview
/>

## 3. 比較運算子

用來比較兩個值：

<TryIt
id="comparison-operators"
starter={`a = 10
b = 5

print(a == b)   # 等於：False
print(a != b)   # 不等於：True
print(a > b)    # 大於：True
print(a < b)    # 小於：False
print(a >= b)   # 大於等於：True
print(a <= b)   # 小於等於：False`}
enableAiReview
/>

## 4. elif：多重條件

處理多種情況：

<TryIt
id="elif"
starter={`score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"

    grade = "D"
else:
    grade = "F"
    
print(f"成績：{grade}")`}

## 5. 邏輯運算子

組合多個條件：

<TryIt
id="logical-operators"
starter={`age = 20
has_license = True

# and：兩個條件都要是 True
if age >= 18 and has_license:
    print("可以開車")

# or：其中一個條件是 True 就可以
weather = "sunny"
if weather == "sunny" or weather == "cloudy":
    print("適合出門")

# not：反轉條件
is_raining = False
if not is_raining:
    print("沒有下雨")`}
enableAiReview
/>

## 6. 巢狀條件

在條件內再加條件：

<TryIt
id="nested-conditions"
starter={`age = 25
has_money = True
movie_rating = "R"

if age >= 18:
    if has_money:
        if movie_rating == "G" or movie_rating == "PG":
            print("可以看電影")
        else:
            print("這部電影不適合")
    else:
        print("沒有錢買票")
else:
    print("年紀太小")`}
enableAiReview
/>

## 7. 簡化的條件表達式

用一行寫簡單的條件判斷：

<TryIt
id="ternary-operator"
starter={`age = 20

# 傳統寫法
if age >= 18:
    status = "成年"
else:
    status = "未成年"

# 簡化寫法
status = "成年" if age >= 18 else "未成年"
print(status)`}
