## 1. 什麼是資料型別？

在 Python 中，每個值都有「型別」，決定了它能做什麼事。
常見型別：

* **整數（`int`）**：沒有小數點的數字
* **浮點數（`float`）**：有小數點的數字
* **字串（`str`）**：用引號包起來的文字

<TryIt
id="data-types"

print(name)`}

## 3. 用 `input()` 讓使用者輸入資料

`input()` 會等使用者輸入一行文字，然後把它存成字串（`str`）。

## 4. 為什麼輸入數字會變成字串？

因為 `input()` **不管輸入什麼都會是字串**。
字串的「加法」會把它們接在一起（串接），而不是數值相加。

<TryIt
id="string-add"
starter={`a = input("輸入第一個數字：")  # 例如 5
b = input("輸入第二個數字：")  # 例如 7
print(a + b)  # 會輸出 57`}
enableAiReview
/>

<TryIt

## 6. 轉換型別（讓數字真的相加）

* `int()`：轉成整數
* `float()`：轉成浮點數
* `str()`：轉成字串

<TryIt
id="type-conversion"
starter={`a = int(input("輸入第一個數字："))
b = int(input("輸入第二個數字："))
print(a + b)  # 這次會正確輸出 12`}
enableAiReview
/>
