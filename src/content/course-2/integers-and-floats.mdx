## 1. 整數（int）vs 浮點數（float）

**整數**：沒有小數點的數字，用來計數或表示完整的量
**浮點數**：有小數點的數字，用來表示更精確的值

<TryIt
id="int-vs-float"
starter={`# 整數
age = 18
count = 100

# 浮點數  
temperature = 36.5
pi = 3.14159

print(type(age))         # <class 'int'>
print(type(temperature)) # <class 'float'>`}

## 2. 數字運算

Python 支援基本的數學運算：

<TryIt

## 3. 除法的兩種結果

* `/`：一般除法，結果一定是浮點數
* `//`：整數除法，結果是整數（向下取整）

<TryIt
id="division-types"
starter={`print(10 / 3)   # 3.3333333333333335
print(10 // 3)  # 3

print(15 / 5)   # 3.0 (還是浮點數！)
print(15 // 5)  # 3 (整數)`}
enableAiReview
/>

## 4. 浮點數的精度問題

電腦儲存浮點數時可能會有微小的誤差：

<TryIt
id="float-precision"
starter={`print(0.1 + 0.2)  # 0.30000000000000004

# 比較浮點數時要小心
result = 0.1 + 0.2
print(result == 0.3)  # False

# 正確的比較方式
print(abs(result - 0.3) < 0.0001)  # True`}
enableAiReview
/>

## 5. 型別轉換

在整數和浮點數之間轉換：

<TryIt
id="number-conversion"
starter={`# 浮點數轉整數（捨去小數）
print(int(3.7))    # 3
print(int(3.99))   # 3

# 整數轉浮點數
print(float(5))    # 5.0

# 字串轉數字
print(int("42"))   # 42
print(float("3.14"))  # 3.14`}
enableAiReview
/>
